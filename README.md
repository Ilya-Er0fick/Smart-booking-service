# Сервис расписаний и встреч

Этот проект представляет собой календарь с возможностью распределения сотрудников банка для встреч с клиентами, 
ориентируясь на свободное время, занятость и район работы сотрудников. Используются для этого локальные базы данных SQL, 
Spring Boot, Java 21.0.1.

## Установка
1. Клонируйте репозиторий:
```
   git clone https://github.com/Ilya-Er0fick/Smart-booking-service.git
   cd Smart-booking-service
```
2. Установите необходимые библиотеки.
3. Скомпилируйте проект.

## Функционал

Через графический интерфейс будет определяться возможность выбора и назначения сотрудника компании для встречи с клиентом,
перед этим у каждого клиента будет несколько входящих параметров: желаемое время и удобный район для встречи. От этих
данных уже будет работать приложение, оно предоставит список всех сотрудников с различным рейтингом. 

### Построение базы данных сотрудников

Каждый сотрудник будет обладать собственным ФИО, ID, доступным и занятым временем, рабочими дням и рейтингом.
